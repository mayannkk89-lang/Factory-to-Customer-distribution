# Factory-to-Customer-distribution
This is my second Git Repository : This project focuses on analyzing and optimizing the shipping routes from factory to customer for Nassau Candy Distributor. By leveraging data analytics, the study identifies inefficiencies in current logistics operations, evaluates delivery lead times, and highlights bottlenecks across regions. The goal is to design smarter, cost-effective, and time-efficient shipping strategies that improve customer satisfaction while reducing operational expenses. Key aspects include:

Route Mapping & Efficiency: Comparing existing factory-to-customer routes to uncover delays and redundancies.
Data-Driven Insights: Using Python, SQL, and visualization tools to analyze shipment data and performance metrics.
Optimization Strategies: Proposing route adjustments, regional aggregation, and streamlined logistics workflows.
Business Impact: Enhancing delivery speed, lowering transportation costs, and strengthening Nassau Candy’s competitive edge.
